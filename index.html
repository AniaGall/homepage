Moje notatki:

Strona internetowa ma warstwy jak cebula i Shrek ;)

- 0 – treść (najważniejsza),
- 1 lub 2 – semantyka: HTML,
- 2 lub 3 – prezentacja: CSS (wygląd),
- 3 lub 4 – interakcja: Java Script.
